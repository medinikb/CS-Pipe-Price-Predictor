Calculation methodology

Weight Method

Pipe mass formula: W = 0.0246615 x (OD - t) x t.

  • W = pipe mass in kg/m.
  • OD = actual outside diameter in mm from the built-in OD table.
  • t = wall thickness in mm entered by the user.
  • Total weight = W x length in meter.

Price Method

  • Raw steel Rs/kg is selected by year unless the user enters Raw Steel Rs/kg Override.
  • Finished Rs/kg = raw steel Rs/kg x estimate factor.
  • Rs/m = finished Rs/kg x pipe kg/m.
  • Total Rs = Rs/m x pipe length.

Factor Method

  • Coating Yes normal factor = 2.30 and P90 factor = 3.80.
  • Coating No normal factor = 1.80 and P90 factor = 2.70.
  • If Estimate Factor Override is entered, it replaces the normal factor.
  • P90 is recalculated using the default P90-to-normal factor ratio.

Coating Definition

  • Internal coating means epoxy lined.
  • External coating means PE coated, or polyethylene coated.
  • Select Coating = Yes for internal coating, external coating, or both.

Reviewer Override Check

Method Used In This Review Version

Weight is calculated using ASME-style pipe mass formula.

  • W = pipe mass in kg/m
  • OD = actual outside diameter in mm
  • t = wall thickness in mm

Price is estimated as raw steel price multiplied by the coating-based finished-pipe conversion factor.

If Estimate Factor Override is entered, it replaces the normal estimate factor. The P90 factor is recalculated using the default P90-to-normal factor ratio.

For this calculator, select Coating = Yes when the pipe has internal epoxy lining, external PE coating, or both. Internal coating means epoxy lined. External coating means PE coated, or polyethylene coated.
